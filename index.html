📖 청소 동선 플래너 사용 설명서

1. 기본 사용 흐름

  1. 출발지 입력 - 집/사무실 등 동선 시작 위치
  2. 출발 시간 설정 - 그 날 처음 나가는 시간
  3. 현장 추가 - 방문할 작업지들을 등록 (주소 + 작업시간)
  4. 도착지 선택 - 귀가/마지막현장/별도지정 중 선택
  5. 최적 동선 계산하기 버튼 클릭 → 결과 확인

2. 요일별 동선 관리

상단 월~일 탭에서 요일별로 다른 동선을 저장할 수 있어요. 오늘 요일에는 빨간 점(●)이 표시되고, 각 요일의 입력값과 계산 결과는 자동으로 저장됩니다. 매주 비슷한 동선을 다니는 경우 편하게 쓸 수 있어요.

3. 혼잡 보정(%)

카카오 모빌리티 API가 알려준 이동시간에 추가 여유를 곱하는 값이에요.

  • 0%: 한산한 시간대, 도로 막힘 없음
  • 30~50%: 출퇴근 시간대 (월~금 7~9시, 17~19시) 추천
  • 50% 이상: 강수/사고 등 특수 상황
예: API가 20분이라고 알려주고 보정 30%면 → 20 × 1.3 = 26분으로 계산

4. 현장별 방문 가능 시간대 (Time Window)

각 현장 카드 아래쪽 방문 가능 시작~종료 시간을 입력하면, 그 시간 안에 도착할 수 있도록 동선을 짭니다.

  • 비워두기: 시간 제약 없음 (아무 때나 방문 가능)
  • 시작만 입력: 그 시간 이후 방문 (일찍 도착하면 대기)
  • 종료만 입력: 그 시간 이전 방문
  • 둘 다 입력: 그 사이에 도착해야 함
⚠️ 시간 제약이 너무 빡빡하면 일부 현장은 방문 불가로 표시되고 카드가 빨갛게 변해요. 그땐 시간대를 늘려보거나 작업시간을 줄여보세요.

5. 🔀 최적 순서 버튼

현장 목록을 가장 효율적인 순서로 자동 재배열합니다.

  • 방문 가능 시간대가 있는 현장 → 마감 시간이 빠른 것부터 우선 배치
  • 같은 우선순위면 → 가까운 곳부터 정렬
  • 너무 일찍 도착하는 경우 → 시작 시간까지 대기 처리

한 번 누르면 현장 목록 순서가 바뀌고, 그 상태로 계산하기를 누르면 새 순서대로 동선이 계산돼요.

6. 📍 지도 추가

버튼을 누르면 지도 클릭 모드가 활성화돼요. 지도 위에서 추가하고 싶은 위치를 클릭하면 그 좌표의 주소가 자동으로 현장으로 추가됩니다. 다시 누르면 종료.

7. 작업시간 기본값

현장 추가 시 작업시간이 자동 설정됩니다:

  • 첫 현장: 30분
  • 두 번째부터: 바로 위 현장과 같은 값

같은 패턴의 작업이 반복될 때 빠르게 입력할 수 있어요.

8. 결과 공유

  • 💬 카카오톡 공유: 카카오톡 친구에게 동선 카드 전송
  • 🔗 링크 복사: URL로 동선을 공유 (받은 사람이 열면 같은 동선이 자동 계산됨)
  • 📋 텍스트 복사: 상세 일정을 텍스트로 복사해 어디든 붙여넣기 가능

9. 자주 묻는 질문

Q. 새로고침해도 입력값이 그대로인가요?
네. 모든 입력값과 계산 결과는 요일별로 브라우저에 자동 저장됩니다.

Q. 한 현장만 시간 제약이 있어요. 가능한가요?
네. 제약이 있는 현장만 입력하면 됩니다. 나머지는 비워두세요.

Q. 계산이 안 되거나 오류가 떠요.
주소가 정확하지 않거나, 시간 제약이 너무 빡빡한 경우입니다. 주소를 검색 자동완성으로 다시 선택해보세요.

Q. 화면을 새로고침해도 최신 버전이 안 나와요.
Ctrl+Shift+R (Mac은 Cmd+Shift+R) 강력 새로고침을 해보세요.
